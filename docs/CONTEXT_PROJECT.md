# CONTEXT_PROJECT.md

## Nome do projeto

Gerenciador de Finanças Pessoal

## Tipo

Aplicação desktop local (sem servidor, sem internet)

## Objetivo

Permitir controle financeiro individual com cadastro de receitas e despesas,
saldo consolidado, histórico e filtros básicos.

## Público

Três perfis principais:

1. Desenvolvedor em formação (Python + GUI + banco de dados)
2. Profissional de outra área usando agentes para construir o sistema
3. Leitor do livro praticando Programação Aumentada por IA

## Stack

Python 3.12 | Tkinter | SQLite

## Escopo atual

MVP local, monousuário, sem autenticação, sem nuvem.

## Princípio de projeto

Simplicidade com coerência entre artefatos.
